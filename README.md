JPZipLocation
=============

An iOS Client for ZipLocate

## Installation

Install with [CocoaPods](http://cocoapods.org):

pod 'JPZipLocation'

## Usage

#### See the example

## Examples

There is an example included with the project. To install, 

```
$ pod install
$ open ZipLocateSample.xcworkspace/
```
